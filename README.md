React Sass Boilerplate
===
Features
---
JavaScript
- Hot reloading
- ES6 with Babel
- ESLint
- Source maps

CSS
- Sass
- Autoprefixer

Setup
---
```
npm install
npm start
# Visit http://localhost:8080
```
